Principais Comandos para Criar um Site

Estrutura Básica do HTML

O HTML (HyperText Markup Language) é a linguagem usada para criar a estrutura básica de um site. Os principais elementos incluem:

- <html>: Define o início e o fim de um documento HTML.
- <head>: Contém metadados do site.
- <body>: Contém o conteúdo visível do site.

Comandos de Estilo com CSS

O CSS (Cascading Style Sheets) é usado para estilizar os elementos de uma página HTML. Os comandos comuns incluem:

- color: Define a cor do texto.
- font-size: Define o tamanho da fonte.
- font-family: Define a fonte do texto.
- background-color: Define a cor de fundo.
